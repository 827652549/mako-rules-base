# 项目词汇表（Next.js）

> 通用术语见 `base/glossary.md`。

## 渲染策略
- **SSR**: Server-Side Rendering — 服务器端渲染，通过 Next.js `async` Server Component 实现
- **CSR**: Client-Side Rendering — 客户端渲染，`'use client'` 组件
- **SSG**: Static Site Generation — 构建时静态生成
- **ISR**: Incremental Static Regeneration — 增量静态再生成，用于半静态内容（`revalidate`）
- **PPR**: Partial Prerendering — 部分预渲染，Next.js 15+ 特性
